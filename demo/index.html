    • One screen scroll Jquery plugin, that lets you create a one page scroll website. The plugin allows you to create beautiful screens for the entire window from one or several sections. You can set the number of sections on the screen, fix the header or footer depending on the width of the window.

    How to install

    This package can be installed with:

    • npm: npm install --save romenko.screenscrooll
    • bower: bower install --save romenko.screenscrooll
    • download from GitHub
    How to use

    How to use

    Put the required stylesheet at the top of your markup:

    <link rel="stylesheet" href="/bower_components/screenScroll/dist/screenscroll.min.css" />
    <link rel="stylesheet" href="/node_modules/screenScroll/dist/screenscroll.min.css" />

    Put the script at the bottom of your markup right after jQuery:

    <script src="/bower_components/screenScroll/dist/screenscroll.min.js"></script>
    <script src="/node_modules/screenScroll/dist/screenscroll.min.js"></script>
    Next

    Document structure

    <header.header>...</header>
<main.main>...</main>
<footer.footer>...</footer>

    Call the plugin function

    $(document).ready(function(){
  $('main.main').screenScroll({
     header: $('header.header'),
     footer: $('footer.footer')
  });
});
    Options

    Options

    headerHeader node
    footerFooter node
    footerFixedFixed footer panel
    headerFixedFixed header panel
    startScreenStart screen position
    scrollTimeOutDelay after move screen
    responsiveResponsive options
    Events

    Events

    screenScroll.initWhen the plugin initializes.
    screenScroll.resizeWhen the plugin gets resized.
    screenScroll.afterMoveAfter screen move.
    screenScroll.beforeMoveBefore screen move.
    screenScroll.nextGoes to next screen
    screenScroll.prevGoes to previous screen
    screenScroll.moveToMove to screen by index
    screenScroll.goToGo to screen by index
    screenScroll.moveToSectionMove to screen where has section index
    screenScroll.moveToSectionIdMove to section where has section id
